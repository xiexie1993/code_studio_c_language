# C语言学习与练习记录项目

## 一、概述



###  1.1、 项目阐述

+ 学习C语言的代码记录
+ 积累代码实例

###  1.1 编写目的

+ 为项目的开发设计的依据与指导
+ 为参与该项目的编程人员提供依据
+ 为修改、维护提供条件
+ 好记性不如烂笔头

#### 1.1.2 读者对象

+ 项目开发人员，特别是编程人员
+ 软件维护人员
+ 技术管理人员
+ 开卷者

####  1.1.3 注意事项

+ 该文档采用 markdown 编写规范，建议使用markdownPad查看和修改
