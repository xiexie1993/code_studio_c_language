### 2、 使用说明

#### 2.1、 步骤总纲

+ 1、 编译生成可执行文件
+ 2、 运行可执行文件

#### 2.2 详细步骤

+ 1、 编译生成可执行文件
    + 方法a：打开命令窗口，切换到当前目录, 执行 gcc main.c -o print_now_time
    + 方法b：执行写好的编译脚本 compile_start.bat（windows）

+ 2、 运行可执行文件
    + windows系统： 打开命令窗口，切换到当前目录, 执行 print_now_time.exe 
    + linux系统：   打开命令窗口，切换到当前目录, 执行 ./print_now_time
